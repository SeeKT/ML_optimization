最適化手法の数値実験例

人工的な関数に対して別資料で示した最適化アルゴリズムを適用する (ここではミニバッチなどは考慮していない)．

以下の3つの関数を用いる．

f1(x,y)=14x2+y2(1)f_1(x, y) = \frac{1}{4}x^2 + y^2 \tag{1}f1​(x,y)=41​x2+y2(1)f2(x,y)=x3+y3−3xy(2)f_2(x, y) = x^3 + y^3 - 3xy \tag{2}f2​(x,y)=x3+y3−3xy(2)f3(x,y)=−4e−(x2+y2)−2e−((x+4)2+(y+4)2)(3)f_3(x, y) = -4 e^{-(x^2 + y^2)} - 2 e^{-((x + 4)^2 + (y + 4)^2)} \tag{3}f3​(x,y)=−4e−(x2+y2)−2e−((x+4)2+(y+4)2)(3)

それぞれに対して，

  1. 最急降下法
  2. Newton 法
  3. Momentum 法
  4. NAG
  5. AdaGrad
  6. RMSProp
  7. Adam

を適用する．ただし，勾配は自動微分を用いて求めることにする．

f1f_1f1​ に対する最適化

  • この関数は凸関数．Newton 法が非常に高速．
  • Momentum や NAG では値の振動が見られる．

f2f_2f2​ に対する最適化

  • Newton 法は2次の最適な方向に進むので，鞍点に陥っている．
  • 簡単な関数形であるということもあり，1次の最適な方向に進み，学習率を Armijo rule で調整した steepest の収束が早い．

f3f_3f3​ に対する最適化

  • 2つの極小値以外の値がほぼ 0 であるような関数であり，通常の steepest の収束は非常に遅い．
  • 一方で，学習率を適合的に調整する3つの手法の収束は早い．
